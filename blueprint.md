# 심리 테스트 애플리케이션 블루프린트

## 1. 프로젝트 개요

따뜻하고 매력적인 UI를 통해 사용자의 성향을 분석하는 심리 테스트 애플리케이션입니다. 20개의 질문에 대한 답변을 바탕으로 8가지 성향 중 하나의 결과를 제공합니다. 구글 애드센스 가이드라인을 준수하여 설계 및 제작됩니다.

## 2. 디자인 및 스타일 가이드

- **컨셉**: 따뜻함, 부드러움, 아늑함
- **색상 팔레트**:
  - 메인 배경: `#FFF5E1` (코스믹 라떼)
  - 주 색상: `#FFDAB9` (피치 퍼프), `#FFA07A` (라이트 살몬)
  - 텍스트: `#5D4037` (다크 브라운)
  - 포인트 색상: `#FFC0CB` (핑크)
- **폰트**: 부드러운 느낌의 웹 폰트 사용 (예: Google Fonts의 'Nanum Pen Script' 또는 'Gaegu')
- **레이아웃**: 중앙 정렬, 모바일 반응형 디자인 (Flexbox 및 Grid 사용)
- **이미지**: 각 성향 결과에 맞는 상업적 무료 이미지 사용 (Unsplash, Pexels 등)

## 3. 기능 명세

### 3.1. 핵심 기능: 심리 테스트
- **질문**: 총 20개의 2지선다형 질문
- **답변 로직**:
  - 3가지 성향 축(사회성, 사고방식, 감정)에 따라 답변을 분류합니다.
  - 각 축에서 더 많이 선택된 경향을 조합하여 8가지 결과 유형 중 하나를 도출합니다.
  - (A/B: 7문제, C/D: 7문제, E/F: 6문제 = 총 20문제)
- **결과**: 8가지 성향 유형별 상세 설명과 관련 이미지 표시

### 3.2. 성향 유형 (8가지)
1.  `ACE`: 현실적인 모험가
2.  `ACF`: 따뜻한 현실주의자
3.  `ADE`: 창의적인 탐험가
4.  `ADF`: 이상적인 중재자
5.  `BCE`: 신중한 전략가
6.  `BCF`: 다정한 현실주의자
7.  `BDE`: 고독한 발명가
8.  `BDF`: 낭만적인 이상가

### 3.3. 페이지 구성 (애드센스 준수)
- **메인 페이지 (`index.html`)**: 테스트 시작 및 진행 화면
- **결과 페이지**: 테스트 완료 후 결과 표시
- **소개 페이지 (`about.html`)**: 사이트 및 테스트에 대한 소개
- **연락처 페이지 (`contact.html`)**: 문의 양식 또는 연락 정보
- **개인정보처리방침 (`privacy.html`)**: 개인정보 수집 및 이용에 대한 안내
- **이용약관 (`terms.html`)**: 서비스 이용에 관한 약관
- **탐색 메뉴**: 모든 페이지로 쉽게 이동할 수 있는 상단 네비게이션 바

## 4. 기술 스택 및 구현 계획

- **Frontend**: HTML, CSS, JavaScript (프레임워크 없이 순수 기술 사용)
- **UI 컴포넌트**: Web Components (Custom Elements)를 사용하여 질문 카드, 결과 페이지 등 재사용 가능한 UI 모듈 생성
- **JavaScript 로직 (`main.js`)**:
  - 질문 데이터 (배열)
  - 사용자 답변 추적
  - 결과 계산 로직
  - 동적 HTML 렌더링 (질문 및 결과)
- **스타일링 (`style.css`)**: 디자인 가이드에 따른 CSS 작성
